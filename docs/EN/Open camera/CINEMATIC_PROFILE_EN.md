# 🎬 CINEMATIC PROFILE — OPPO Reno 11F (Open Camera)

## 📷 1. BASIC SHOOTING MODE

### ✔ Resolution
* 4K (if stable)
* otherwise 1080p 60fps (better for motion and smoothness)

### ✔ Video bitrate
* 4K → 50–80 Mbps
* 1080p → 20–35 Mbps
👉 Important: don't set it to "Auto" — it often ruins gradients

## 🌈 2. COLOR AND EXPOSURE

### ✔ Scene mode
* Auto
* or HDR ON (if there's a separate switch)

### ✔ Exposure compensation (EV)
* Daylight: -0.3
* Contrasty Sun: -0.7
* Clouds/Fog: 0
👉 OPPO always slightly reduces the light — we replicate this manually.

### ✔ ISO
* Auto (with a max ISO 800 limit)
or manual:
- Day: 100–200
- Evening: 400–800

### ✔ Shutter (if manual)
* 1/60 → Universal
* 1/120 → Bright Day/Motion
* 1/30 → Evening (but requires a stabilizer)

## 🎨 3. COLOR (MOST IMPORTANT)

### ✔ White Balance
Auto — Normal
or Fixed:
```Text
| Scene | WB |
| ------------ | ---------- |
| Daylight | 5200–5600K |
| Sunset | 6000–6500K |
| Lamps | 3200–4000K |
```

👉 Fixed WB = less "spots" and color jumps

## 🧠 4. REMOVING THE MAIN PROBLEM (COLOR SPOTS)

If available:
### ✔ Noise reduction
Low / Off (Low is better for video)

### ✔ Edge enhancement
Low

### ✔ HDR algorithm (if available)
Auto HDR / Multi-frame HDR
👉 If you enable aggressive denoise → those "color spots" appear

## 🎥 5. SMOOTH LIKE OPPO

### ✔ Frame rate
* 30 fps → cinematic
* 60 fps → documentary / action
👉 recommendation:
* base: 4K 30 fps
* dynamics: 1080p 60fps

## 🔒 6. IMAGE STABILITY

### ✔ Focus mode
* Continuous video focus (Caf)
or
* Locked focus (if the scene is static)
👉 OPPO often “smoothly focuses” without twitching – we avoid jumps

## 🧪 7. OPPO LOOK PSYCHOLOGY (Important)

OPPO does 3 things that you simulate manually:
* ↓ slightly underexposes
* ↑ gently stretches the dynamic range
* ↓ smooths out noise without destroying color
👉 therefore:
* don't overexpose
* don't raise the ISO
* don't use aggressive denoise

## 🎬 READY-MADE “PRESET BY MIND”

Simplifying it down to One rule:
> "Better a slightly darker, but pure gradient than a bright one with color spots"

## 💡 FOR DOCTOR FILM

This profile will give you:
* a more "lively" image without plastic
* smooth transitions between the sky and skin
* less post-processing in Shotcut
* a more "cinematic" source
